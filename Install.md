## ⚙️ Installation

Questa guida descrive come configurare l’ambiente per eseguire l’applicazione in locale.

---

### 1. Clonare il repository

```bash
git clone <repository-url>
cd FinanziAI
```

---

### 2. Creare ambiente virtuale (consigliato)
```bash
python3 -m venv venv
source venv/bin/activate  	# Linux / Mac
venv\Scripts\activate		# Windows
```

---

### 3. Installare le dipendenze
```bash
pip install -r requirements.txt
```

---

### 4. Esecuzione 
```bash
python main.py
```
